# C Practice Projects

This repository contains a collection of simple C programs I’ve written while learning the language using the BroCode tutorial: https://www.youtube.com/watch?v=87SH2Cn0s9A.  
Each project focuses on practicing core programming concepts such as variables, loops, conditionals, functions, arrays/2D arrays, and input/output.  

## Projects

- **Basic Bank Account Program**  
  A simple program that simulates deposits, withdrawals, and displaying an account balance.
  Practices: Taking users input and using functions to update a balance (withdraw or deposits) and displaying it.

- **Basic Calculator Program**  
  Performs basic arithmetic operations (addition, subtraction, multiplication, division).
  Practices: Basic input/output from a user and the math functions found in math.h.

- **Circle Circumference Program**  
  Calculates the circumference of a circle based on a given radius.
  Practices: Basic input/output from a user.

- **Hypotenuse Calculator Program**  
  Uses the Pythagorean theorem to calculate the hypotenuse of a right triangle.
  Practices: Basic input/output from a user, as well as, using math functions built into math.h.

- **Movie Seating Project**  
  Prints out a seating chart for a movie theatre and allows seat booking.
  Practices: 2D arrays (printing and updating the cells).

- **Temperature Conversion Program**  
  Converts temperatures between Celsius and Fahrenheit.
  Practices: Basic input/output from a user.

Purpose:
The goal of this repository is to practice and improve my skills in C programming, starting from the basics and working toward more advanced projects over time to prepare for my Introduction to Programming course in the Fall 25 semester at the University of Guelph.
