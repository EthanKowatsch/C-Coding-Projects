# C Practice Projects

This repository contains a collection of simple C programs I’ve written while learning the language.  
Each project focuses on practicing core programming concepts such as variables, loops, conditionals, functions, arrays/2D arrays, and input/output.  

## Projects

- **Basic Bank Account Program**  
  A simple program that simulates deposits, withdrawals, and displaying an account balance.  

- **Basic Calculator Program**  
  Performs basic arithmetic operations (addition, subtraction, multiplication, division).  

- **Circle Circumference Program**  
  Calculates the circumference of a circle based on a given radius.  

- **Hypotenuse Calculator Program**  
  Uses the Pythagorean theorem to calculate the hypotenuse of a right triangle.  

- **Movie Seating Project**  
  Prints out a seating chart for a movie theatre and allows seat booking.  

- **Temperature Conversion Program**  
  Converts temperatures between Celsius and Fahrenheit.  

Purpose
The goal of this repository is to practice and improve my skills in C programming, starting from the basics and working toward more advanced projects over time.
